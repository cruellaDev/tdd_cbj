# 대역

## 대역의 목적
1. 테스트 대상이 외부 요인에 의존하여 테스트가 어려울 경우  대역을 써서 테스트 진행
   - 외부요인은 테스트 작성을 어렵게 만들 뿐만 아니라 테스트 결과도 예측할 수 없게 만든다.
       - 테스트 대상에서 파일 시스템을 사용
       - 테스트 대상에서 DB로부터 데이터를 조회하거나 데이터를 추가
         - 상황에 맞는 테이블이나 데이터를 구성할 수 없는 경우
       - 테스트 대상에서 외부의 HTTP 통신
         - 외부 API 서버가 일시적 장애
